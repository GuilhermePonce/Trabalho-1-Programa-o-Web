Trabalho 1 para a disciplina de INF1407

Descrição do projeto:
Implementar sem javascript uma operação web com todas as operações CRUD.
O projeto combinado era de realizar um meeting planner, onde é possível um gerente marcar reuniões com seus funcionários e seus funcionários responderem se estarão presentes ou não.
As resposabilidades de cadastro foram colocadas no gerente por combinado.
